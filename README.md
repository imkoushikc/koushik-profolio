# Koushik Chaturvedula — Personal Portfolio

Personal portfolio website built with React + Vite.

## Tech Stack
- React 18
- Vite
- Pure CSS animations
- Canvas API (particle background)
- Google Fonts (Syne + DM Mono)

## Getting Started

```bash
# Install dependencies
npm install

# Run locally
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Deployment

This project auto-deploys to GitHub Pages via GitHub Actions on every push to `main`.

Live site: https://imkoushikc.github.io/koushik-profolio/

Manual deploy:
```bash
npm run deploy
```

## Customization

All content is in `src/Portfolio.jsx` inside the `DATA` object at the top of the file. Update:
- Personal info
- Work experience
- Projects
- Skills
- Social links
